# 职位薪资影响因素分析

这是回归分析课程的大作业项目，主要分析职位薪资数据及其影响因素。

## 项目结构

```
project/
├── README.md          # 项目说明文档
├── main.rmd          # 主文档，整合所有分析
├── src/             # 源代码目录
│   ├── preprocessing.rmd  # 数据预处理
│   ├── description.rmd    # 描述性统计
│   └── regression.rmd     # 回归分析
└── data/            # 数据目录
    ├── data_origin.xlsx   # 原始数据
    └── data.xlsx          # 处理后的数据
```

## 分析内容

1. 数据预处理
   - 数据清理
   - 对数变换
   - 类别变量处理

2. 描述性统计
   - 因变量分析
   - 自变量分析
   - 变量关系分析

3. 回归分析
   - 模型构建
   - 变量选择
   - 模型诊断
   - 结果解释
